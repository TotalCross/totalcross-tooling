# Repository instructions for coding agents

Fabio Sobral is the sole current maintainer. Read `.agent/PLANS.md` and any
applicable nested instructions before editing. Keep an active ExecPlan current
for long or multi-step work.

Preserve imported authors and history. Never remove third-party notices or add
obsolete contact email addresses to current content. Run focused checks before
broad builds, keep verbose output in files, and summarize it concisely.

During migration, avoid cross-plugin refactoring. Do not rewrite history,
force-push, delete tags, or archive repositories without explicit scope and
verified backups.
