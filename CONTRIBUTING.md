# Contributing

Keep project-specific changes within the relevant subdirectory and use that
project's existing build tooling. Before opening a change, run the root
validation command:

    python3 tools/check-license-headers.py

Repository-wide governance changes must preserve historical attribution and
license notices. Fabio Sobral ([@flsobral](https://github.com/flsobral)) is the
sole current maintainer and default code owner.
