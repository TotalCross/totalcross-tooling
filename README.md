# TotalCross Tooling

A monorepo for the TotalCross Maven plugin, VS Code extension, and Gradle
plugins. Each project keeps its own build, versioning, and release lifecycle.

- [`maven-plugin/`](maven-plugin/)
- [`vscode-extension/`](vscode-extension/)
- [`gradle-plugin/`](gradle-plugin/)

Run `python3 tools/check-license-headers.py` to validate repository governance.
